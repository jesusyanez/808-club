<h2>Features</h2>

* Download one of multiple drumkits
* Automatic file unzipping
* Downloads to selected folder
* Supports all operating systems
* Made with Python 3</br>


### Downloads from:
- [X] Google Drive
- [X] Dropbox
- [X] MediaFire
- [X] WeTransfer
